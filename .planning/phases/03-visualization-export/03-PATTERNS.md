#### `@assert` vs `@inbounds` usage

**Analog:** `src/energia.jl` lines 28–30 (`oblicz_macierz_dystans!`) and `src/baselines.jl` lines 51–53:

```julia

    for i in 1:j-1
```

```julia

@assert 1 <= start <= n "start out of range"

```

**Apply to `wizualizacja.jl`:** `wizualizacja.jl` is NOT a hot path — omit `@inbounds`. The helper `_trasa_do_punkty` can use `@inbounds` if it loops over `stan.trasa` (N=1001 elements per frame). Use `@assert` with English messages for internal invariant checks per LANG-04. Avoid `@assert` for user-facing file-path checks (use `isfile()` + `error()`).

**Source:** Every existing `src/*.jl` file — universal project convention.
**Apply to:** All new/modified files in Phase 3.
